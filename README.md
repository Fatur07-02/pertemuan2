## 🛠️ Tech Stack
- **Framework:** React Native (Expo SDK 50)
- **Navigation:** Expo Router
- **Language:** TypeScript

## 🚀 Cara Menjalankan
1. Clone repository ini.
2. Jalankan `npm install`.
3. Jalankan `npx expo start`.
